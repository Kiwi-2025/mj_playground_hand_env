## XML 文件的导入顺序与最后的关节顺序紧密相关
如果你导入顺序为 hand -> cube 那么，hand的关节会cube之前；如果顺序为 cube -> hand 那么 cube 的关节会在 hand 之前。
这是因为 MuJoCo 在加载 XML 时会按照顺序分配关节 ID，所以导入顺序会影响关节的最终排列顺序，请务必注意这一点。无论添加什么道具，都要确保手部模型始终是最开始导入的部分，以保证手部关节在关节数组中的正确位置。

## 异步阻塞错误
你可能会在训练过程中观察到一些异步阻塞错误。这通常是由于渲染线程和物理计算线程之间的同步问题引起的。不用担心，这是正常现象，不会影响训练过程。
