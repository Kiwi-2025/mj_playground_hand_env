## tmux指令
- 新建会话： `tmux new -s <session_name>`
- 分离会话： Ctrl + b 然后按 d
- 列出会话：` tmux ls`
- 重新连接会话： `tmux attach -t <session_name>`
- 关闭会话： `exit` 或 Ctrl + d （在会话内）
- 杀死会话： `tmux kill-session -t <session_name>`
- 杀死全部会话： `tmux kill-server`
- 上下翻滚解锁 : Ctrl + b 然后按 [ ，退出翻滚模式按 q
